Proyectos

Proyecto Login con PHP

Creé un sistema de login con PHP y MySQL donde los usuarios se registran e inician sesión de forma segura. Usé una base de datos para almacenar sus datos y, al iniciar sesión, PHP verifica las credenciales y mantiene al usuario autenticado mediante sesiones. También implementé medidas de seguridad para proteger la información.

CRUD con PHP y MySQL

Creé un CRUD con PHP y MySQL que permite realizar operaciones básicas (crear, leer, actualizar y eliminar) en una base de datos. Los usuarios pueden agregar, ver, editar y eliminar registros a través de formularios, y PHP maneja la interacción con la base de datos para actualizar los datos en tiempo real.

App ÁgilMente

Desarrollé Ágilmente, una aplicación para ayudar a personas mayores a trabajar el deterioro cognitivo a través de ejercicios que estimulan la memoria y concentración, con una interfaz sencilla y adaptada a sus necesidades.
